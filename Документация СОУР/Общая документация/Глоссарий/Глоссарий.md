## Основные термины

| Термин                                        | Сокращение | Определение                                                                                                                                                                                |
| :-------------------------------------------- | :--------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Разрешительный документ                       | РД         | Документ, определяющий содержание, место, время, условия безопасного выполнения работ, состав бригады и ответственных лиц. Типы: наряд-допуск, распоряжение, поручение, работы по перечню. |
| Силы и средства                               | СиС        | Модуль СОУР для управления штатными и командированными бригадами, учёта персонала и техники.                                                                                               |
| Техническое место                             | ТМ         | Объект электросетевого хозяйства (линия, подстанция, участок), на котором выполняются работы.                                                                                              |
| Единица оборудования                          | ЕО         | Конкретное оборудование, установленное на техническом месте.                                                                                                                               |
| Мобильное устройство / Мобильное приложение   | МУ / МП    | Планшет или смартфон на ОС Android (или Аврора) с установленным приложением СОУР для работы полевых бригад.                                                                                |
| Система управления производственными активами | СУПА (SAP) | Мастер-система на базе SAP (или аналога), содержащая заказы ТОРО, НСИ, данные по оборудованию, персоналу и материалам.                                                                     |
| Нормативно-справочная информация              | НСИ        | Справочники системы (виды работ, типы заказов, ключи образца, дефекты, материалы), загружаемые из СУПА.                                                                                    |
| Электронная цифровая подпись                  | ЭЦП        | Используется для подтверждения этапов РД (инструктажи, выдача, допуски).                                                                                                                   |
