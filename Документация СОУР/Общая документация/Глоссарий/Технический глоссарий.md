## Роли в бригаде

| Термин | Сокращение | Определение |
| :--- | :--- | :--- |
| Ответственный руководитель работ | ОРР | Лицо, отвечающее за безопасность и организацию работ мобильной бригадой (как правило, мастер). |
| Производитель работ | ПР | Член бригады, непосредственно руководящий выполнением работ на объекте. |
| Допускающий | – | Лицо из числа оперативного персонала, осуществляющее подготовку рабочего места и допуск бригады к работе. |
| Наблюдающий | – | Лицо, назначаемое для надзора за бригадой при работах в особо опасных условиях; при его наличии назначение других членов бригады (кроме ПР) часто ограничено. |
| Член бригады | staff | Рядовой участник мобильной бригады. |

## Интеграционные и технические термины

| Термин | Сокращение | Определение |
| :--- | :--- | :--- |
| Ключ образца | KTSCH | Технический идентификатор операции в SAP. **Критичный параметр:** если он отсутствует в данных операции, факт выполнения работ не будет принят мастер-системой при отправке из СОУР. |
| Код операции | ACTIVITY | Интеграционный параметр, определяющий действие с объектом: 1 — создание, 2 — обновление, 3 — отзыв или удаление. |
| Деблокирован | Released | Статус заказа в SAP (СУПА), обязательный для того, чтобы заказ стал видимым и доступным для выбора в СОУР. Заказы в статусе «Технически закрыт» или «Только создан» не отображаются. |
| Объект не из справочника | custom_object | Техническое место или единица оборудования, введённые пользователем вручную текстом. Для таких объектов недоступна загрузка схем из СУПА и автоматическая подтяжка операций. |
| Локальное выполнение | – | Режим работы с РД (преимущественно для подрядчиков), при котором данные вводятся вручную в веб-интерфейсе без использования мобильного приложения и ЭЦП. Продление, перерывы и перевод на другое рабочее место не фиксируются. |
| Мягкое удаление | Soft delete | Метод удаления данных (например, шаблонов бригад), при котором запись не стирается из БД физически, а помечается флагом `deleted_soft`. Запись остаётся в БД для истории и аналитики, но перестаёт отображаться в интерфейсе. |
| Геофенсинг / Радиус работ | – | Ограничение в мобильном приложении, блокирующее прохождение этапов РД, если GPS-координаты устройства находятся дальше заданного расстояния от объекта. |
| Суббригада | – | Временная группа сотрудников, автоматически выделяемая системой из состава штатной бригады при их распределении по разным разрешительным документам. |

## Смежные системы и протоколы

| Термин | Сокращение | Определение |
| :--- | :--- | :--- |
| Active Directory | AD | Корпоративная служба каталогов, используемая в СОУР для аутентификации пользователей (единый доменный пароль). |
| Wialon | – | Мастер-система GPS-мониторинга, предоставляющая данные о местоположении транспорта и пробеге для формирования путевых листов. |
| Журнал учета работ | ЖНДР | Электронный журнал учета работ по нарядам и распоряжениям, интегрируемый с системой СК-11. |
| Очередь отправки | Dispatch | Технические таблицы в БД (например, `electrical_diagram_dispatch`), хранящие задания на передачу данных в шину ЕИП. |
