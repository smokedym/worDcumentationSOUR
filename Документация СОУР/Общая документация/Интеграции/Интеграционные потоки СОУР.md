## Потоки взаимодействия с СУПА (SAP)

| Поток           | Направление | Назначение                                                                                                     | Протокол           | Особенности                                                                |
| :-------------- | :---------- | :------------------------------------------------------------------------------------------------------------- | :----------------- | -------------------------------------------------------------------------- |
| П.5 / П.20      | СУПА → СОУР | Передача плановых заказов ТОРО, операций, материалов.                                                          | SOAP/XML через ЕИР | Инициатор — SAP (транзакция `ZPM_ORD_SOURSENT`)                            |
| П.7 (факт)      | СОУР → СУПА | Передача факта трудозатрат, физических объёмов, израсходованных материалов.                                    | SOAP/XML           | Данные уходят при нажатии кнопки «Передать в СУПА» в Вебе                  |
| П.7 (транспорт) | СОУР → СУПА | Передача заявок на транспорт (создание и отказ).                                                               | SOAP/XML           | Требует наличия роли «Заказчик» (Выдающий) в бригаде                       |
| П.8 (транспорт) | СУПА → СОУР | Передача согласованных заявок на транспорт.                                                                    | SOAP/XML           | Асинхронный ответ. Обновляет данные в РД или даёт подсказку                |
| П.16            | СУПА → СОУР | Передача НСИ: справочники (виды работ, типы заказов, ключи образца, дефекты, единицы измерения, склады и др.). | SOAP/XML           | Еженедельное обновление по регламенту (00:00 МСК)                          |
| П.17            | СУПА → СОУР | Передача иерархии объектов: технические места (ТМ) и единицы оборудования (ЕО).                                | SOAP/XML           | Обновление по заводу автоматически 1 раз в неделю                          |
| П.18            | СУПА → СОУР | Передача признаков объектов эксплуатации (напряжение, марка и т.д.).                                           | SOAP/XML           | Данные используются для формирования точек измерений.                      |
| П.19            | СУПА → СОУР | Передача топологии электрической сети (связи между объектами).                                                 | SOAP/XML           | Используется для визуализации схем и фильтрации ТМ в РД.                   |
| П.21            | СУПА → СОУР | Передача остатков материалов на складах.                                                                       | SOAP/XML           | Запрос из СОУР выполняется каждую минуту по активным складам.              |
| П.23            | СУПА → СОУР | Передача точек измерений для ТМ и ЕО.                                                                          | SOAP/XML           | Формирует основу для протоколов испытаний на МУ.                           |
| П.24            | СУПА → СОУР | Передача дефектов объектов эксплуатации.                                                                       | SOAP/XML           | Фильтруются дефекты со статусами СООТ, СОРА и ЗКПР.                        |
| П.28            | СОУР → СУПА | Запрос и получение схем электроустановок.                                                                      | SOAP/XML           | Доступно для ТМ из справочника в статусах от «Планирование» до «В работе». |

## Потоки взаимодействия с другими системами

| Поток           | Направление                             | Назначение                                                                  |
| :-------------- | :-------------------------------------- | :-------------------------------------------------------------------------- |
| П.13            | 1С:ЗУП → СОУР                           | Передача кадровой информации о сотрудниках (приём, увольнение, должность).  |
| П. трудозатраты | СОУР → 1С:ЗУП                           | Передача данных о трудозатратах для расчёта зарплаты.                       |
| П.2-5 (СК-11)   | СК-11 ↔ СОУР                            | Интеграция ЖНДР и заявок на отключение (только для Нарядов и Распоряжений). |
| П.9-11 (1С:ПБ)  | 1С:Производственная безопасность → СОУР | Передача данных по охране труда: права НДС, обучение, СИЗ.                  |
| П.14            | СОУР → РГИС                             | Передача маршрутов перемещения бригад (в целевой модели).                   |
| П.14/ Wialon    | W → РГИС                                |                                                                             |
